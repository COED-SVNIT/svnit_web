  • Introduction: Hardware

    • (06 Hours)

    Introduction to embedded systems Hardware needs; typical and advanced, timing diagrams, memories (RAM, ROM, EPROM) Tristate devices,    Buses, DMA, UART and PLD's Built-ins on the microprocessor.

  • Interrupts

    • (03 Hours)

    Interrupts basics ISR; Context saving, shared data problem. Atomic and critical section, Interrupt latency.

  • Software and OS

    • (03 Hours)

    Survey of software architectures, Round Robin, Function queue scheduling architecture, Use of real time operating system. RTOS, Tasks, Scheduler, Shared data reentrancy, priority inversion, mutex binary semaphore and counting semaphore.
  • IPC

    • (05 Hours)

    Inter task communication, message queue, mailboxes and pipes, timer functions, events Interrupt routines in an RTOS environment.

  • Software design

    • (06 Hours)

    Embedded system software design using an RTOS Hard real-time and soft real-time system principles, Task division, need of interrupt routines, shared data.

  • Tools

    • (14 Hours)

    Embedded Software development tools. Host and target systems, cross compilers, linkers, locators for embedded systems. Getting embedded software in to the target system, Debugging techniques. Testing on host machine, Instruction set emulators, logic analyzers In-circuit emulators and monitors.
  • Introduction to Smart Card Technology
