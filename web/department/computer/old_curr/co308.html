  • Introduction to the unix,  Systems Standards and Implementations

    • (03 Hours)

    Introduction to Unix system, review of the basic concepts of Files & Directories, Logging in, Input & Output, Programs & Processes, error handling, System calls and Library functions, Unix Standardization, Limits, primitive system data types.
  • File Input /Output & Related System calls

    • (05 Hours)

    Introduction to the Files in UNIX - File Descriptors, File Handle Representation,  System calls for File Input & Output, I/O efficiency, File Sharing.
  • Files and Directories

    • (09 Hours)

    File types in UNIX, System calls and their use for non-regular types of files, Set-User-ID and Set-Group-ID, File Access Permissions,  Concept of Sticky bit,  File size, File Truncation, Symbolic Links. File Times, Directory Handling & Manipulations, Special Device files, Summary of File Access Permission Bits.
  • System Data Files, System Information

    • (05 Hours)

    Introduction, Password File,  Shadow Passwords,  Group Files,  Other Data files used by the system to store vital information,  Login Accounting Files, System Identification, Time and Date Routines.

  • The Environment of Unix process 

    • (05 Hours)

    The Environment of a Unix Process, main function,  Process termination functions,  Environment Lists,  Memory Layout of a C Program,  Shared Libraries,  Automatic, Register and Volatile Variables.
  • Process control

    • (10 Hours)

    Introduction, Identifiers, Creating processes, File Sharing, Race conditions, Execution, Changing user IDs and Group Ids, Interpreter files, Set-User-ID, User Identification, Process Times

  • Inter Process Communication

    • (05 Hours)

    Introduction, Pipes, Coprocesses, FIFOs, Message Queues, Semaphores, Shared Memory, Client Server Properties.

    (Total Contact Time: 42 Hours)

    PRACTICALS

    1). Implement cat command.
    2). Implement link, unlink, and rename, symlink, readlink system calls.
    3). Implement who command.
    4) .Print all environment variables.
    5). Write a program that creates a zombie and verify that the process is zombie.
    6). Send a data from parent to child over a pipe.
